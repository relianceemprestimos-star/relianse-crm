# Plano de refatoracao controlada

## Objetivo

Reduzir atrito operacional sem reescrever o CRM inteiro.

## Fase 1 — organizacao segura

- Menu lateral simplificado.
- Campanhas como centro operacional.
- Tema claro/escuro.
- Credenciais como ponto oficial de logins de averbadores.
- Documentacao das regras.

## Fase 2 — migracao operacional

- Relatorio de bases/campanhas sem grupo.
- Associar historicos de margem aos grupos corretos.
- Criar filtros mensais por campanha.
- Padronizar nomes de convenios e orgaos.

## Fase 3 — automacao guiada

- Importar base fria dentro da campanha.
- Rodar margem automaticamente conforme convenio.
- Enviar somente positivos elegiveis para telefone.
- Aguardar coeficientes quando necessario.
- Liberar selecao para vendedor ou disparo controlado.

## Freios

- Nao apagar dados.
- Nao quebrar Nova Vida, CapSolver ou averbadores.
- Nao enviar WhatsApp sem opt-in ativo.
- Nao publicar sem validacao.
